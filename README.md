# Banking System Project
A banking system with a VB.NET GUI interface and C++ backend for core banking operations.

## Features
- Create, view, and manage accounts
- Deposit and withdraw funds
- Check balance and transaction history
- User-friendly VB.NET interface

## Tech Used
- VB.NET (GUI)
- C++ (Backend logic)

## How to Run
1. Clone this repo
2. Open in Visual Studio
3. Build and run the VB.NET project
